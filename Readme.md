# livekit-client-cpp-sdk

C++ client SDK for LiveKit in C++17.

## Support Platforms

- [x] Microsoft Windows
- [ ] GNU/Linux
- [ ] Apple macOS
- [ ] iOS
- [ ] Android

## Dependencies

- [livekit-protocol](https://github.com/livekit/protocol)(as submodule by default)
- [plog](https://github.com/SergiusTheBest/plog) (as submodule by default)
